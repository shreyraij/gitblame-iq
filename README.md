# GitBlame IQ 🧠

> Turn raw git history into developer personality profiles.

When you join a new codebase, `git blame` tells you *who* wrote a line — but not *who to ask*, *who owns what*, or *how each developer thinks*. GitBlame IQ analyzes a repository's full commit history and builds a profile for every contributor.

## What it does

- **Ownership maps** — which modules/files each developer dominates
- **Coding style fingerprints** — comment density, avg function complexity, commit size patterns
- **Commit rhythm analysis** — time-of-day, day-of-week, burst vs steady patterns
- **Natural language summaries** — plain English profile per developer powered by Claude AI
- **HTML report** — shareable, self-contained report you can open in any browser
